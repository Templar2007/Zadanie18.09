ID Identyfikator, unikalny atrybut przypisany do elementu w dokumencie HTML, który służy do identyfikowania tego elementu w kontekście stylowania CSS oraz manipulowania nim za pomocą JavaScript. ID to sposób, aby jednoznacznie zidentyfikować i odnaleźć określony element na stronie internetowej. Umożliwia to dostęp do elementu w celu zmiany jego wyglądu, zachowania lub zawartości.
Cel stosowania ID Stosowanie ID ma na celu jednoznaczne identyfikowanie konkretnych elementów na stronie, co umożliwia precyzyjne manipulowanie nimi przy użyciu styli CSS oraz skryptów JavaScript. ID pozwala na wyjątkowe oznaczenie elementów, co ułatwia zarządzanie nimi i manipulację nimi za pomocą kodu.
Znaczenie znaku # Znak "#" jest używany przed nazwą ID w arkuszu stylów CSS lub jako atrybut "id" w elemencie HTML do wskazania, że identyfikator jest używany. # oznacza, że następna nazwa jest identyfikatorem, który ma być używany do jednoznacznego określenia danego elementu.
Wywołanie ID Wywołanie ID następuje poprzez przypisanie identyfikatora do odpowiedniego elementu w dokumencie HTML za pomocą atrybutu "id". Przykładowo: <p id="pochyly">Tekst</p> - oznacza, że ten paragraf ma identyfikator "pochyly".
Uwagi na temat stosowania ID Różnica między ID a class polega na tym, że identyfikator może występować tylko raz w całym dokumencie, natomiast klasę można powielać do woli. Elementy oznaczone ID muszą być unikatowe na stronie, a ich nadmiarowe używanie jest błędem. W przypadku wątpliwości, czy dany element powinien mieć unikalne oznaczenie, lepiej użyć klasy, która może być przypisana wielu elementom.
